# Modern Personal Portfolio Website

A colorful, animated, and responsive personal portfolio website built with modern web technologies.

## Features

- Modern, colorful design with gradient effects
- Responsive layout for all device sizes
- Smooth animations and transitions
- Dark mode toggle
- Scroll-triggered animations
- Interactive elements and hover effects
- SEO-friendly structure

## Technologies Used

- HTML5
- CSS3 (with animations and gradients)
- JavaScript (ES6+)
- TypeScript
- Bootstrap 5
- Vite
- AOS (Animate On Scroll)
- GSAP (GreenSock Animation Platform)
- Font Awesome

## Sections

1. Hero Section
2. About Me
3. Experience
4. Skills
5. Education
6. Projects
7. Contact
8. Footer

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Aftab272/portfolio-website.git
   ```

2. Navigate to the project directory:
   ```bash
   cd portfolio-website
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

### Development

To start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:5173` (or the next available port).

### Build

To create a production build:
```bash
npm run build
```

The build files will be generated in the `dist` directory.

### Preview

To preview the production build locally:
```bash
npm run preview
```

## Deployment

This project is configured for deployment on Vercel. Simply connect your GitHub repository to Vercel and it will automatically deploy on every push to the main branch.

## Customization

To customize the portfolio with your personal information:

1. Update the content in `index.html`
2. Replace placeholder images with your own
3. Modify the contact information
4. Update social media links
5. Adjust colors in `src/style.css` if desired
